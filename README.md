# TO-DO-LIST-PYTHON
--------To-Do List (Python)---------

A clean and simple command-line To-Do List project built using Python.
This project helps you manage your daily tasks easily.

---

Features:

-  Add new tasks
-  View all tasks
-  Delete tasks
-  Mark tasks as complete
-  Edit tasks
-  Saves tasks using JSON file

---

Technologies Used:

- Python
- JSON (for data storage)

---

How to Run:

1. Make sure Python is installed
2. Download or clone this repository
3. Run the file:

python your_file_name.py

---

File Structure:

project/
│
├── tasks.json      # Stores tasks
└── main.py         # Main program

---

Example:

========= TO DO LIST =========
1. Add Task
2. View Tasks
3. Delete Task
4. Mark Complete
5. Edit Task
6. Exit

Purpose:

This project is created for learning Python basics like:

- File handling
- Functions
- Lists & dictionaries
- User input
